# Blazor-Hadmin

本项目使用**Blazor  WebAssembly**  作为客户端，案例前后的分离





## 技术架构

- C#
- dotnet8
- ant design ui
- webapi
- mongodb



## 目录结构

| 目录   | 描述                             |
| :----- | -------------------------------- |
| font   | 前端 blazor Webassembly          |
| api    | 后端代码（本案例未实现使用可忽略 |
| shared | 共享数据模型                     |
