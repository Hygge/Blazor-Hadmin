# Blazor-Hadmin

本项目使用**Blazor  WebAssembly**  作为客户端，案例前后的分离

**🚀直接拉下代码即刻运行🚀**



## 技术架构

- C#
- dotnet8
- ant design ui
- webapi
- mongodb



## 目录结构

| 目录   | 描述                             |
| :----- | -------------------------------- |
| font   | 前端 blazor Webassembly          |
| api    | 后端代码（本案例未实现使用可忽略） |
| shared | 共享数据模型                     |

### 项目实现技术点
- AuthenticationStateProvider 授权登录
- Ant Design Blazor 组件库的路由复用多标签页，端tabs多个标签页功能这个比较麻烦研究过很久
